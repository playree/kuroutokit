Message

メッセージを表示するためのコンポーネントです。
Class
kk-message メッセージ
Attribute
指定なし Infomationの配色
theme="success" Successの配色
theme="warning" Warningの配色
theme="error" Errorの配色

Message

メッセージ

メッセージ表示です。kk-messageを指定してください。 
<div class="kk-message kk-margin-top">
  Infomation message.
</div>
<div class="kk-message kk-margin-top" theme="success">
  Success message.
</div>
<div class="kk-message kk-margin-top" theme="warning">
  Warning message.
</div>
<div class="kk-message kk-margin-top" theme="error">
  Error message.
</div>
