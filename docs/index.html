KUROUTO Kit
サポートブラウザ Chrome Firefox Edge
最新バージョン0.0.2


KUROUTO Kitは、製作者の好みを大いに盛り込んだCSSフレームワークです。
ですので、おそらくはかなりとんがった仕様になっているのではないかと。(笑)

もし気に入ってくれる方がいれば、ご自由にお使いください！
国産CSSフレームワークとして少しでも使っていただけたら嬉しいです。
CSS定義はまだまだ少ないですが、徐々に追加していく予定です。

※いまだ開発途中なので、大幅に仕様が変更される可能性があります。
　定義がある程度まとまり、大枠の仕様が固まった時点で v1.0.0 としてリリースします。

Installation

使い方

<!-- KUROUTO Kit -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/playree/kuroutokit@0.0.2/dist/kuroutokit.min.css" />
<script src="https://cdn.jsdelivr.net/gh/playree/kuroutokit@0.0.2/dist/kuroutokit.min.js"></script>

ローカルに取り込む場合は、GitHubのリリースページよりダウンロードすることができます。

Feature

特徴

  • 各種CSS定義にカスタムプロパティを採用。
    カスタムプロパティを上書きすることでデザインのカスタムが容易に。
    フレームワークを使うことによりデザインが似通ってしまうデメリットの対策にも。
  • 基本フォントにWebフォントを採用。
    各種デバイスでフォントを統一。(もちろん、通常フォントも使えます。)
  • クラス名はkk-*で統一し、クラス名の衝突を回避。
  • クラスで基本コンポーネントを指定し、大きさやデザインなどの違いを属性で指定する形の設計としています。
  • 日本語のドキュメント。（整備中ですが。。。
  • おそらく、既存CSSフレームワークとしてUIkitの影響を一番受けています。
  • もちろん、このページもKUROUTO Kitで作成しています。
    ページの編集にはHTML Worksを使用しています。（さりげなく宣伝
