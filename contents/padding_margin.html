Padding & Margin

パディングやマージンの定義です。

Padding

パディング

パディングの設定です。
レスポンシブの記述が利用できます。
Class
kk-padding-ss padding: 0.25rem
kk-padding-s padding: 0.5rem
kk-padding padding: 1.0rem
kk-padding-l padding: 2.0rem
kk-padding-rm padding: 0
kk-padding*@s 640px 以上の場合のみ有効
kk-padding*@m 960px 以上の場合のみ有効
kk-padding*@l 1200px 以上の場合のみ有効
Attribute

Margin

マージン

マージンの設定です。
Class
kk-margin-top-ss margin-top: 0.25rem
kk-margin-top-s margin-top: 0.5rem
kk-margin-top margin-top: 1.0rem
kk-margin-top-l margin-top: 2.0rem
kk-margin-left-ss margin-left: 0.25rem
kk-margin-left-s margin-left: 0.5rem
kk-margin-left margin-left: 1.0rem
kk-margin-left-l margin-left: 2.0rem
kk-margin-rm margin: 0
kk-margin-top-rm margin-top: 0
kk-margin-bottom-rm margin-bottom: 0
kk-margin-right-rm margin-right: 0
kk-margin-left-rm margin-left: 0
