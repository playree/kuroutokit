Accordion

アコーディオンのように、閉じたり開いたりするためのコンポーネントです。
Class
kk-accordion アコーディオン
Attribute

Accordion

アコーディオン

アコーディオン表示です。kk-accordionを指定してください。
直後に記載の要素を開いたり閉じたりできるようになります。
下記サンプルの場合はid="view"のdiv要素がアコーディオンの対象になります。 
<div class="kk-accordion kk-font-bold" theme="icon">Accordion</div>
<div id="view">
  <div class="kk-margin-top">↑をクリックすることで表示を隠すことができます。</div>
  <input id="input1" class="kk-input kk-margin-top" type="text" value="INPUT" placeholder="Input">
  <button id="button1" class="kk-button kk-margin-top" type="submit">Button</button>
</div>
